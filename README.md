# netfree-www-pages


## Requirements
  * [node](http://nodejs.org) >= 0.10 
  * git

## Installation
    $ git clone git@github.com:magicode/netfree-www-pages.git
    $ cd netfree-www-pages
    $ npm i
  
## Build
    $ cd netfree-www-pages
    $ node build
  
## Preview
    $ cd netfree-www-pages
    $ node test-server

  browse to http://localhost:3000/
